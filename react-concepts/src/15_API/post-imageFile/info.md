# ⭐ 1. FormData:- Browser ka built-in object jo file ya form data ko API me send karne ke liye use hota hai.

👉 Use:- Image upload, Video upload, PDF upload, Multiple files

👉 Kaise kaam karta hai:- File ko multipart/form-data format me backend ko send karta hai.

👉 Best For:-
✅ Real projects
✅ Large files upload

<!--  -->

# ⭐ 2. Base64:- Image ya file ko string (text format) me convert karta hai.

👉 Use:- Image preview, Profile avatar, Small images

👉 Problem:-
❌ Size 30-35% bada ho jata hai.

👉 Best For:-
✅ Small image
✅ Preview show karna

<!--  -->

# ⭐ 3. Multer :- Backend middleware (Node.js + Express) jo FormData se aaye files ko receive aur save karta hai.

👉 Use:-

- Server folder me image save karna.
- Multiple image upload handle karna.

👉 Best For:-
✅ Backend file handling.
